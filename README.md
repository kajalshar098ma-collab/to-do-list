# Advanced To-Do App

A modern and responsive To-Do App built using:

- HTML
- CSS
- JavaScript

This project helps users manage daily tasks with advanced features like:

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Dark Mode
- Search Tasks
- Filter Tasks
- Task Categories
- Due Dates
- Drag & Drop Reordering
- Local Storage Support

---

# Features

## Add Tasks
Users can add new tasks easily.

## Edit Tasks
Tasks can be updated anytime.

## Delete Tasks
Remove unwanted tasks instantly.

## Complete Tasks
Mark tasks as completed.

## Dark Mode
Switch between Light and Dark themes.

## Search Functionality
Search tasks in real-time.

## Filters
Filter tasks by:
- All
- Completed
- Pending

## Categories
Organize tasks into categories:
- Personal
- Study
- Work
- Shopping

## Due Dates
Assign deadlines to tasks.

## Drag & Drop
Reorder tasks by dragging.

## Local Storage
Tasks remain saved after page refresh.
